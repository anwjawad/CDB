Upload the coordination tracker Excel file (.xlsx) to load or refresh patient data. The file is parsed entirely in your browser — nothing is sent to any server.

The workbook must contain a sheet named "Tracking sheet" with a header row that includes a "patient" column, and a "Lists" sheet for dropdown reference data.
