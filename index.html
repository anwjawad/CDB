OneDrive Tracker Sync

This dashboard opens with the configured OneDrive sharing link, downloads the latest workbook, parses it locally using SheetJS, and refreshes automatically every 5 minutes while the page is open.

For automatic sync, the OneDrive link must allow direct browser download. If Microsoft returns a permission page, use an "Anyone with the link can view" sharing link or upload a local copy.
